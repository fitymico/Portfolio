Большинство облачных PostgreSQL (Heroku, Supabase, Yandex Managed) требуют SSL. Добавьте в connection string параметр sslmode=require:
  • Откройте Источники → PostgreSQL → Настройки
  • В поле Connection URL допишите ?sslmode=require
  • Если используете self-signed — sslmode=verify-ca + загрузите CA-сертификат
