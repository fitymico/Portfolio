## 📝 Заметки

- Для всех heavy-WebGL и SPA-сайтов (Bruno Simon, Apple Vision Pro, Active Theory, Resn) **обязательно открыть в браузере вживую** — статичный fetch не показывает суть.
- При просмотре открывай **DevTools → Network** на каждом — чтобы понять какие библиотеки используются (Lenis / GSAP / Three.js / Locomotive). Это ускоряет реверс-инжиниринг.
- Хорошие источники для дальнейшего поиска: **awwwards.com**, **godly.website**, **minimal.gallery**, **siteinspire.com**.
